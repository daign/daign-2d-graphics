# daign-2d-graphics

[![NPM package][npm]][npm-url]

Two dimensional graphics library that implements the [daign-2d-pipeline][pipeline-url].

## Installation

```sh
npm install @daign/2d-graphics --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
